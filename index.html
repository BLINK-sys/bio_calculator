📊 Основная формула расчета

(X / 1.2 × курс × 1.12 + доставка) × 1.18

💡 Важно: Для расчета доставки используется максимальный вес из фактического веса товара и объемного веса (объем × 200)

📋 Формулы доставки БИО (Москва → Астана):

До 30 кг: 7500 + 10000 + (26×400) + 4000 = 31,900 тг
30-300 кг: 7500 + (вес-30)×179 + 10000 + (вес-30)×20 + 10400 + 4000 + (вес-30)×15
300-1000 кг: Сумма 3 компонентов (город+город, забор склад БИО, доставка по Астане)
Свыше 1000 кг: Используется формула для 1000 кг
